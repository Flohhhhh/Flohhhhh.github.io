Generator Pseudocode

- Generate colors from input color
- Set "colors" array to generated colors
- Create particles at random location with random color from "colors" array
- Move particles with Perlin Noise Flow Field
- Draw lines with particles based on location
- Redraw background at low opacity to make lines appear to have depth

How to use Generator

The generator will automatically draw constantly. In order to change the colors and drawing style of the generator, you can manipulate the sliders below it, and then press Q to reset the background.

Controls

  • Q = Reset Background and change color
  • Color Picker = Background Color
  • Slider 1 = RED
  • Slider 2 = GREEN
  • Slider 3 = BLUE
  • Slider 4 = ALPHA
  • Slider 5 = Weight

If you want to save the image, simply right click the canvas.
